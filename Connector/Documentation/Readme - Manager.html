MF SQL Context Menu - Readme

The files to configure the MFSQL Context Menu have been installed. The install folder includes:

  1. M-Files Replication files to import the Context Menu structure and metadata components.
  2. M-Files Vault application
  3. M-Files UI extensibility application

The Context Menu is designed as a building block to configure custom SQL procedure calls from within M-Files using one of the following methods:

  1. Menu item in custom window to trigger SQL Procedure
  2. Context sensitive menu item to trigger a SQL Procedure for the object
  3. Workflow state action plug in the call a SQL Procedure

Follow the instructions outlined in Installing the Context Menu guide to complete the configurations and installation.

Consult the Online Documentation to use the Connector and its components.

The key configuration steps

  • Import the M-Files Replication to add the structure and related metadata
  • Install both M-Files Applications
  • Update connectionstring in the configuration object
  • Add menu items to the database table MFContextMenu
  • Add custom procedures for menu operations

The Connector is designed as a developer tool to design and build integrations with M-Files and to do extensive metadata manipulation. There is much more to it.
