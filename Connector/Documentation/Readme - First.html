MFSQL Connector Installer

MFSQL Connector consist of several modules. These modules must be installed on the targeted server. Each installation package is self standing and can be downloaded to a folder location that is accessible to the appropriate servers. Execute in sequence from the server to install.

If SQL, M-Files and IIS are all installed on the same server, the packages are all installed on that server.

  1. SQL Database Server: MFSQL Connector Core
  2. M-Files Server: MFSQL Conetext Menu
  3. Web Server: MFSQL Connector Core

Follow Installation, upgrades and patches for further guidance during installation.
