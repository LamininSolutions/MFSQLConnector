MF SQL Connector Readme

Congratulations! You have successfully installed MFSQL Connector into the database. This installation created tables, views and procedures in the target database. Other extensions builds onto the Connector. Install the other packages into the designated servers to access the functionality of these extensions. The core extensions include:

  1. Context Menu - This extension allows MFSQL Connector operations from within context on M-Files Desktop Client UI

Consult the Online Documentation to use the Connector and get details about each procedure and steps.

First steps

Use SSMS or Visual to explore the Connector Database. The MFSQL Manager will also be very handy to explore the metadata in depth

There are many different operations and procedures that can be performed with the Connector, however the most basic steps to get started include the following initialization of the Connector by using the following procedures

  • Test vault connection: exec spMFVaultConnectionTest
  • First metadata syncronization: exec spMFSynchronizeMetadata
  • Create first class table: exec spMFCreateTable "Customer"
  • Get records for the table from M-Files: exec spMFUpdateTable "MFCustomer",1

The Connector is designed as a developer tool to design and build integrations with M-Files and to do extensive metadata manipulation. There is much more to it.
